# 今日の問題 - インタラクティブなカウンター

`problem/index.html`を開き、`problem/script.js`を編集して、カウンター機能を完成させてください。

## 完成形

- `+` ボタンをクリックすると、数字が1増えます。
- `-` ボタンをクリックすると、数字が1減ります。
- `リセット` ボタンをクリックすると、数字が0に戻ります。

## ファイル構成

- `index.html`: カウンターの画面です。
- `style.css`: 画面のスタイルシートです。
- `script.js`: カウンターのロジックを記述するJavaScriptファイルです。**このファイルに追記・修正を加えてください。**

## 解答の確認方法

1. `problem/index.html` をブラウザで開きます。
2. `script.js` の `// ここにコードを書いてください` と書かれている部分を埋めて、カウンターの機能を完成させます。
3. ブラウザをリロードし、各ボタンが正しく動作することを確認します。

## テストの実行方法

このプロジェクトでは、`vitest` を使用して、`script.js`のロジックをテストします。

1.  まず、必要なパッケージをインストールします。`20251112`ディレクトリで、ターミナルで以下のコマンドを実行してください。
    (初回のみで大丈夫です)

    ```bash
    npm install
    ```

2.  次に、以下のコマンドでテストを実行します。

    ```bash
    npm test
    ```

    テストがすべて通れば、カウンターのロジックは正しく実装されています。
