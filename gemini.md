このプロジェクトは，デイリーで javascript+html+css の問題を解くものです．

# 必須事項

respose は必ず日本語で返してください．

# 背景

私は javascript+html+css を勉強中です．
一日に一問 javascript+html+css を使った問題を解きたいです．
その問題を作成して下さい．

# 要件

一日ごとにフォルダを作成します．
例）20251111

そのフォルダ内には，

- problem（問題）
- test（テスト）
- correct（解答）
  を作成してください．

# problem

problem は問題用フォルダです．
その日の問題が格納されています．

問題形式は穴埋め形式で出してください．
穴埋めの割合は６割くらいでお願いします．

使う package は自由ですが，使用の際は使い方を載せてください．

# test

vitest を使った test フォルダを作成してください．

problem フォルダ内にそのテストの仕方，手順も載せてください

# correct

問題の解答用フォルダです．
この中に正解と格納してください．
