# 今日の問題

`index.html`を開き、`style.css`を編集して、ボックス内のテキストを赤色にしてください。

## ファイル構成

- `index.html`: 問題のHTMLファイルです。
- `style.css`: スタイルを記述するCSSファイルです。このファイルに追記・修正を加えてください。
- `script.js`: JavaScriptファイルです。今回は使用しません。

## 解答の確認方法

1. `problem/index.html` をブラウザで開きます。
2. `style.css` を編集し、指定された要素の文字色を赤色に変更します。
3. ブラウザをリロードすると、文字色が赤色に変わっていることが確認できます。

## テストの実行方法

このプロジェクトでは、`vitest` を使用してテストを実行します。

1.  まず、`vitest` をインストールします。ターミナルで以下のコマンドを実行してください。
    (初回のみで大丈夫です)

    ```bash
    npm install
    ```

2.  次に、以下のコマンドでテストを実行します。

    ```bash
    npm test
    ```

    今回はCSSの見た目に関する問題なので、テストは自動化されていません。
    テストは`vitest`のセットアップが正しく行われているかを確認するためのものです。
